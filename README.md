## Usage

If you want to run the Python 3 implementation, I have a script (`pyaoc25.py`)
that will run any day on any input. Use the command:

```sh
$ python pyaoc25.py <day> <input_path>
# Example command
$ python pyaoc25.py 1 inputs/day01.txt
```
